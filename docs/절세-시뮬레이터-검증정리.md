> 절세 제안 6종의 **개별 정확성·중복 여부**를 검증하고, 고객에게 **합산 절감액**을
> 정직하게 보여주기 위한 **우선순위 규칙**을 정리했습니다. 결론과 의사결정 포인트는 맨 아래.

## 1. 기준 세금 계산 검증 — 통과 ✅

금융소득종합과세 세액(절세 효과의 기준선)을 표준 공식과 대조:

- **우리/PR30**: `전체 금융소득 × 15.4%(원천징수) + 초과분 × (한계세율 − 15.4%)`
- 표준 근사(`2,000만 × 15.4% + 초과분 × 한계세율`)와 **모든 케이스 일치**.

> 참고: 비교한 타 팀(everyyess)은 `초과분 × (한계 − 15.4%)`만 적용해 **초과분의 원천징수 15.4%를 누락** → 과소 계산. 우리 방식이 표준대로 맞음.

## 2. 6종 개별 절감액 검증 — 전부 통과 ✅

각 전략의 코드 절감액 = 원리원칙(세전 − 세후) 계산과 일치 (시나리오: 금융소득 5,000만·한계 46.2%):

| 전략 | 절감액 | 비고 |
|---|---|---|
| 1. ISA | 310.2만 | ISA는 종합과세 합산 제외 가정 |
| 2. 연금 | 118.8만 | **세액공제(다른 메커니즘)** — 산출세액이 공제액 이상이어야 전액 효과 |
| 3. 분리과세채 | 132.0만 | **한계세율 > 33%일 때만** 효과 |
| 4. 저율배당 | 369.6만 | 보수적(종합과세 추가분만 회피) — 과소=안전 |
| 5. 해외 250만 공제 | 55.0만 | 연 양도차익 ≥ 250만이어야 만액 |
| 6. 손익통산 | 110.0만 | 보수적(250만 공제 중복효과 미반영) |

→ **공식은 모두 정확하고, 보수적 가정은 전부 "덜 잡는" 방향이라 과장 없음.**

## 3. 중복(상호작용) 문제 — 단순 합산은 과대평가 ⚠️

6종을 **각자 같은 기준에 대고 독립 계산**하므로, 단순 합산하면 **같은 과세 풀을 중복 차감**함.

**공유 풀(겹치는 자원):**
- **종합과세 초과분(excess)** ← 1.ISA · 3.분리과세채 · 4.저율배당 이 함께 사용
- **해외 양도차익** ← 5.해외공제 · 6.손익통산 이 함께 사용
- **2.연금**만 독립 (납입액 기준)

**정량 예시** (금융소득 4,000만·초과분 2,000만·한계 46%, 채권1,500/배당1,500/기타1,000만):

| | 단순 합산 | 순차 결합 |
|---|---|---|
| 합계 | 1,254만 | 930.6만 |

→ **단순 합산이 실제 결합 절감액보다 약 35% 과대평가.** (시나리오·순서에 따라 변하지만, 겹친다는 구조 자체는 항상 성립.)

## 4. 권고 — 우선순위 기반 "결합 절감액"

합계를 보여주되(설득력 ↑) 과대평가를 없애려면, **공유 풀을 한 번씩만 깎도록 우선순위대로 순차 적용**한다.

**우선순위 = 공유 풀 1원당 절감 효율 순:**

- 종합과세 초과분 풀:
  1. **ISA** (≈ 한계 − 9.9% = 약 36%) — 가장 효율 높음 (단, 연 2,000만 한도·3년 의무보유)
  2. **저율배당** (한계 − 15.4% = 약 31%)
  3. **분리과세채** (한계 − 33% = 약 13%, 한계>33%일 때만)
- 해외 양도차익: 5·6은 경쟁이 아니라 실제 양도세 공식상 **함께 차감(스택)** → `절감 = 22% × min(차익, 손실 + 250만)`
- 연금(2): 독립으로 합산

이 순서는 **정확하면서 동시에 절감액을 최대화**한다(효율 높은 전략을 먼저 적용).

**표시 방식:** 카드에 "단독 절감액"이 아니라 **"결합 적용 시 기여분"**을 표기 → 카드 합 = 정직한 총액. (예: ISA 310.2 / 저율배당 369.6 / 분리과세채 0 / 양도세 132 / 연금 118.8 = **930.6만**) 기여 0이 되는 전략은 "이미 다른 전략으로 절세 한도 소진"으로 설명.

> 참고: PR #30에 이미 우선순위 인프라가 있음 — `ISA_PRIORITY_ASSETS`, `IRP_PRIORITY_ASSETS`, `allocate_account_buckets`(계좌 배분 우선순위). 이 위에 "결합 절감액 계산"만 얹으면 됨.
